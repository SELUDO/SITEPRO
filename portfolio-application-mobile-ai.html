Site WEB pour une ludothèque

Ce projet a été parmis les premiers projets que j'ai réalisé. C'est un site WEB pour une Ludothèque. Les clients avaient déjà une identité graphique, je me la suis appropriée et je leur ai designer ce site. Une fois quelques modifications apportées, j'ai pu passer un développement.

Mes clients recherchaient un site "vitrine" sur lequel l'utilisateur pouvait trouver toutes les informations nécessaires : services, horaires d'ouverture, adresse, tarifs et une partie destinée aux contacts.

Technologies utilisées : HTML, CSS
